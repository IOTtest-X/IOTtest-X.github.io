域名、SSL 与备案那些事

打算自己搭一个网站，会遇到三个绕不开的问题：域名怎么注册、HTTPS 怎么配置、国内服务器要不要备案。这篇文章把这三件事捋一遍。

域名注册

域名是你网站的地址，比如 example.com。注册域名需要每年付费，常见后缀的价格差异很大：.com 大约每年 10–15 美元，.cn 在国内平台大约 29–79 元，而 .top、.xyz 这类新后缀首年可能只要几块钱，但续费价格要看清楚。

推荐在 Namecheap、Cloudflare Registrar 或国内的阿里云、腾讯云注册。Cloudflare 的优势是以成本价出售域名，没有额外利润加成。

SSL 证书（HTTPS）

现在几乎没有理由不用 HTTPS。Let's Encrypt 提供免费的 DV 证书，有效期 90 天，可以自动续期。如果你用 Vercel、Netlify 或 Cloudflare 部署网站，他们会自动帮你配置好证书，完全不需要手动操作。

自己管服务器的话，用 Certbot 工具可以一条命令搞定：

certbot --nginx -d yourdomain.com

付费的 OV/EV 证书主要用于银行、支付等对信任要求极高的场景，普通网站没有必要。

ICP 备案

如果你的服务器在中国大陆，网站必须完成 ICP 备案才能正常访问。备案本身免费，但流程繁琐，需要提交域名证书、服务器证明、个人或企业材料，审核周期通常 1–4 周。

如果想绕开备案，可以把服务器放在香港或海外，但访问速度可能受影响。用 GitHub Pages、Vercel 等境外平台托管静态网站是个不需要备案的好选择。
